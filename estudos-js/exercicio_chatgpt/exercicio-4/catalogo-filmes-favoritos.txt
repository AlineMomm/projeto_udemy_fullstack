EXERCÍCIO DE HTML, CSS E JAVASCRIPT

TEMA DO SITE:
Meu Catálogo de Filmes

NÍVEL:
Básico para intermediário

OBJETIVO DO EXERCÍCIO:
Criar um site para organizar filmes em duas seções principais:

1. Filmes assistidos
2. Quero assistir

A página principal será uma tela de catálogo, com cards de filmes lado a lado. O cadastro de filmes será feito em uma segunda página.

O foco do exercício é praticar:

HTML:
- estrutura de página
- header
- main
- section
- cards
- links entre páginas
- formulários

CSS:
- flexbox
- grid
- cards lado a lado
- botões com imagens
- filtros escondidos
- responsividade
- paleta de cores própria

JavaScript:
- eventos
- click
- load
- submit
- input
- change
- arrays
- objetos
- localStorage
- filtros
- renderização dinâmica com innerHTML
- atualização de contadores

17. OBJETO FILME

Cada filme deve ser salvo como um objeto dentro de um array.

Exemplo de filme assistido:

{
    id: 1,
    nome: "Interestelar",
    genero: "ficcao",
    status: "assistido",
    nota: "5",
    prioridade: "",
    comentario: "Filme sobre espaço, tempo e emoções."
}

Exemplo de filme que quero assistir:

{
    id: 2,
    nome: "Duna",
    genero: "ficcao",
    status: "quero-assistir",
    nota: "",
    prioridade: "alta",
    comentario: "Quero assistir porque gosto de ficção científica."
}

18. ARRAY PRINCIPAL

No JavaScript, todos os filmes devem ficar em um array:

let filmes = [];

Esse array será salvo no localStorage.

--------------------------------------------------

19. LOCALSTORAGE

Como o cadastro acontece em uma página diferente, você precisa usar localStorage.

Ao cadastrar um filme:

1. Buscar os filmes já salvos
2. Converter para array
3. Adicionar o novo filme
4. Salvar novamente no localStorage
5. Voltar para index.html

Exemplo de lógica:

const filmesSalvos = localStorage.getItem("filmes");

if (filmesSalvos) {
    filmes = JSON.parse(filmesSalvos);
}

filmes.push(novoFilme);

localStorage.setItem("filmes", JSON.stringify(filmes));

window.location.href = "index.html";

20. RENDERIZAÇÃO NA PÁGINA PRINCIPAL

Quando a index.html carregar:

1. Buscar os filmes no localStorage
2. Separar os filmes assistidos
3. Separar os filmes que quero assistir
4. Renderizar os cards nas seções corretas
5. Atualizar os contadores

Exemplo de separação:

const filmesAssistidos = filmes.filter(function (filme) {
    return filme.status === "assistido";
});

const filmesQueroAssistir = filmes.filter(function (filme) {
    return filme.status === "quero-assistir";
});

Onde usar:
index.html e cadastro.html

O que faz:
- mostrar data atual
- carregar filmes do localStorage
- renderizar cards na tela

--------------------------------------------------

click

Onde usar:
- botão de filtro
- botão Novo Filme
- botão Excluir
- botão Marcar como Assistido
- botão Mover para Quero Assistir

O que faz:
- abrir e fechar filtros
- trocar de página
- excluir filme
- alterar status do filme

--------------------------------------------------

submit

Onde usar:
formulário de cadastro

O que faz:
- impedir o recarregamento da página
- validar campos
- cadastrar filme
- salvar no localStorage
- voltar para index.html

--------------------------------------------------

input

Onde usar:
campos de pesquisa dos filtros

O que faz:
- filtrar os filmes enquanto o usuário digita

--------------------------------------------------

change

Onde usar:
- selects dos filtros
- select de status no cadastro

O que faz:
- aplicar filtros
- mostrar campo Nota ou Prioridade no formulário

--------------------------------------------------

focus

Onde usar:
campos do formulário de cadastro

O que faz:
- destacar o campo selecionado

--------------------------------------------------

blur

Onde usar:
campos obrigatórios do formulário

O que faz:
- validar campo vazio quando o usuário sair dele

--------------------------------------------------

mouseover

Onde usar:
cards dos filmes

O que faz:
- destacar o card ao passar o mouse

--------------------------------------------------

mouseout

Onde usar:
cards dos filmes

O que faz:
- remover destaque do card

--------------------------------------------------

dblclick

Onde usar:
cards dos filmes

O que faz:
- marcar ou desmarcar filme como favorito

Esse recurso pode ser desafio extra.

RESUMO DO EXERCÍCIO

Você vai criar um site chamado Meu Catálogo de Filmes.

A página principal terá duas seções:

- Filmes assistidos
- Quero assistir

Cada seção terá:
- botão de filtro
- botão Novo Filme
- cards de filmes

O botão de filtro abre as opções de filtro daquela seção.

O botão Novo Filme leva para uma nova página de cadastro.

O cadastro salva o filme no localStorage e depois o filme aparece automaticamente na seção correta da página principal.
